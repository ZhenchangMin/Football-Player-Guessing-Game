📖 游戏说明

系统从数据库中随机挑选一位现役球员作为谜底，你需要在有限次数内猜出他是谁。

颜色提示

  • 绿色 该属性与谜底完全一致
  • 黄色 该属性与谜底接近或同类
  • 红色 该属性与谜底不符

各列说明

  • 年龄 / 号码 / 身价：绿色=完全相同；黄色=差一岁/一号/15%以内；红色并显示 ↑↓ 箭头提示方向
  • 位置：绿色=完全相同；黄色=同一位置线（前锋线 / 中场线 / 后卫线 / 门将）
  • 俱乐部 / 联赛：只有完全一致才绿色
  • 国籍：绿色=同国；黄色=同洲；红色=不同洲

位置代码对照

GK 门将 CB 中后卫 LB / RB 左/右后卫 LWB / RWB 左/右翼卫 DM 后腰 CM 中场 LM / RM 左/右中场 AM 前腰 LW / RW 左/右边锋 SS 影子前锋 ST / CF 中锋

游戏设置

  • 可在上方"游戏设置"中调整难度（次数）
  • 可通过身价滑块筛选球员范围，例如只猜身价 €50M 以上的球星
  • 设置在下一局开始时生效
